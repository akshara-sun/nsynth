# About
This project was inspired by the AKAI MPK MINI MKII. I want to challenge myself to not just build a regular piano but a workstation + an instrument. 

# Features
- Mini piano (1 octave) 
- Drum Pads
- Octave Toggle (go up or down the octave)
- Chord recognition 
- Sequencer 
- Regular Size Piano
- Music Notation
